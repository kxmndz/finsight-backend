# Finsight AI Backend

### Installation

Install Node dependencies
```sh
npm install
```

[Install MongoDB](https://www.mongodb.com/docs/manual/installation/)  
make sure `mongod` is in PATH!  

### Setup

#### Mongo
```sh
mongod --dbpath $FINSIGHT_PATH/mongo

```

#### Backend
```sh
npm run start
```

#### Frontend
```sh
npm run dev
```
